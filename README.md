## Getting Started
Checkout this repo, install dependencies, then start the process with the following:
```
> git clone https://github.com/dnisdv/vanillajs-components-notepad
> cd vanillajs-components-notepad
> npm install
> npm run dev
```

## Build the app
```
> npm run build
```
